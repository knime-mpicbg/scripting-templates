Scripting Template Gallery

This page lists all scripting templates available for the various scripting nodes in Knime.

Name Preview Description Category Author Language
estimate gaussion intersection compute the histograms of random variables, fit a gaussian and compute the intersection points. (use the python figure template with the same name to produce the control plots) statistics Felix Meyenhofer Python
Normal distribution tester Tests each group of a numerical column for normality. The null hypothesis at confidence level alpha holds if H=1 and is rejected if H=0 statistics/univariate Felix Meyenhofer Matlab
ANOVA test Tests each group of a numerical column for normality. performs balanced one-way ANOVA for comparing the means of two or more groups, where each group represents an independent sample containing mutually independent observations. statistics/univariate Felix Meyenhofer Matlab
Outlier removal Calculates the outliers for each group it find in the grouping column. A row is removed if it contains an outlier for any of the specified parameter. The method whisker uses the "boxplot-technique" to determine outliers. The remaining data is contained in [q25%-1.5*iqr...q75% + 1.5*iqr]. The meanStd removes all the points further than 3SD from the mean. The remaining data is contained in [mean-3*SD...mean+3*SD] pre-processing Felix Meyenhofer R
Power transformation and Z rank transformation Power transformation (Box-Cox, Yeo-Johnson or basic) can be applied within a specified group of data. Rank normalization (project_to_Gaussian) projects each group of values per parameter to a normal distribution. - select type of transformation - select a group column (e.g. barcode, date, project name,...) - select parameters that should be transformed utilities ?, Martin Stoeter R
Correlation (parameters) Calculates the correlation between all chosen parameters. screenmining Martin Stoeter R
Correlation (profiles) Calculates the correlation between multi-parameteric profiles. screenmining Martin Stoeter R
Correlation (replicates) Calculates the correlation between replicates for all chosen parameters. screenmining Martin Stoeter R
Mahalanobis Distance Calculator Calculates mahalonobis distances to the mean of a control group (typically your negative controls). screenmining null R
Create column name table from selection Creates a table with a single column that contains the selected column headers. - select or type name of column for column headers (TDS internal) utilities Holger Brandl, Martin Stoeter R
Factorized rank transform Adds a new column which contains the rank of a particular item (Order Inductor) within a given group (Group by). utils Holger Brandl R
Reshape to long format Creates a table with a single measurement column and several id columns. utils null R
Unique values Returns unique values for all selected columns independently. If they differ in length, missing values will be added. utils Antje Niederlein R
Connectivity Filter Adds an addtional colum which specifies whether molecules are connected or not. You may want to adapt the name of the string-column which contains the chemical structure. Chemoinformatics Holger Brandl Groovy
